# WebStudio

**WebStudio** is a modern, responsive landing page for a creative agency. This
project focuses on clean semantic markup and high loading speeds.

## 🛠 Technologies & Approaches

- **HTML5**: Semantically correct document structure.
- **CSS3 (BEM)**: Styled using the BEM methodology, ensuring scalability and
  clean code.
- **Vanilla JavaScript**: Used purposefully and exclusively for modal window
  logic (minimal JS footprint).

## ✨ Key Features

- **Responsive Design**: The website is fully optimized for mobile devices,
  tablets, and desktops.
- **Performance First**: Zero heavy libraries or frameworks, resulting in
  top-tier Google Lighthouse scores.
- **Interactive Modals**: A lightweight solution for user interaction without
  the need for third-party scripts.
